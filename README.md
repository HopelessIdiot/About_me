# About_me
telegram bot for sharing information bout yourself

Бот предназначен для работы на хостинге Amver, пно при этом можно его переделать под работу через чисто терминал

вам нужно устоновить Python и к нему библеотеку aiogram.

-----------Структура-----------

MainFolder
├── Main.py              # Код
├── Texts.txt            # Тексты команд
├── Config.txt           # Настройки
├── requirements.txt     # Что нужно для проекта на хосте
└── amvera.yml           # Тут нстройки для хостинга

-----------Как текст менять-----------

Заходите в Texts.txt и прописываете текст сколько вам угодно. Пример оформления:

s<KeyWord>
Меня зовут Name, мне int летку
e<KeyWord>

Где KeyWord будет назвнием команды, а знак пред "<>" озночает: "s" - начало текста "e" - конец текста. Текст начинает с первого правого символа  от "s<KeyWord>" включительно и последним будет самый левый символ от "e<KeyWord>".

Количество команд не ограничено возможностями кода, он превращет в команды ВСЕ такие тексты.

-----------Как конфиг менять-----------

Заходите в конфин и вписываете настройки по строчкам. Пример:

<P>
False  # --Включена ли приватность
w # --режим работы, w/b
@nien,@notnien # -- список пользователей через зпятую

<M>
@nien,@notnien # -- список админов через зпятую

Где <P> - настройки приватности, а <M> - базовые. "w" - белый список, т.е. только указаные могут. "b" - черный список, т.е. только  указаные не могут
